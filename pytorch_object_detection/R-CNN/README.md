### 本地编译文档

需要预先安装以下工具：

```
$ pip install -r requirements.txt
```

### 文档浏览

有两种使用方式

1. 在线浏览文档：[R-CNN](https://r-cnn.readthedocs.io/zh_CN/latest/)

2. 本地浏览文档，实现如下：

    ```
    $ git clone https://github.com/zjZSTU/R-CNN.git
    $ cd R-CNN
    $ mkdocs serve
    ```
    启动本地服务器后即可登录浏览器`localhost:8000`
